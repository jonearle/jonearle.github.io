This assignment builds upon assignment 2 by enchancing the visuals with CSS. 
It removes some of the buttons and JS on the website from Assignment 2 because it did not align with the 
visual aesthetic and professionalism of the site (for example the color changing buttons at the bottom). 
The website contains three sites: index, skills, work, life and feedback. Index is the home page which allows 
for navigation to all other sites. Skills and work are very similar in design. One page shows off my projects 
and skills while the other shows off my work experience and education. Life page is a gallery of some of my 
favourite recent photos. Feedback is a form which, with a proper backend, should allow the user to provide me 
feedback on the site and send it to me directly. However, the main purpose of this was to provide a form and 
style it according to the requirements. Overall, the site is a casual, basic and minimalistic portfolio page 
based on my resume and style taste. 

Hosted on jonearle.github.io !
